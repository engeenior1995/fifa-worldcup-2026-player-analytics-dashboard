# fifa-worldcup-2026-player-analytics-dashboard
interactive Streamlit dashboard analyzing 54,600+ player-match records from FIFA World Cup 2026 across 1,248 players and 48 teams. Features a full data cleaning pipeline, EDA, WebGL-powered 3D visualizations via Three.js and Plotly, correlation analysis, KPI cards, dynamic filters, and CSV export. Built with Python, Pandas, Plotly.
